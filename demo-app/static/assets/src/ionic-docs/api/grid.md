The grid is a powerful mobile-first flexbox system for building custom layouts.

It is composed of three units â€” a grid, [row(s)](row.md) and [column(s)](col.md). Columns will expand to fill the row, and will resize to fit additional columns. It is based on a 12 column layout with different breakpoints based on the screen size. The number of columns can be customized using CSS.

See the [Responsive Grid documentation](/docs/layout/grid) for more information.
